# ETH Taipei Website

This is the repo of ETH Taipei official website.
ETH Taipei is a local ETH community that hosts the annual ETHTaipei conference and regular monthly events in Taipei, Taiwan.
Welcome to drop by [events](https://ethtaipei.org/#events) we host if you're around :)

## Commands

- dev mode: `npm run dev`
- production preview:

```
npm run build
npm run start
```

## Deployment

is on Vercel; contact ETH Taipei team for more info.
